Устанавливаете NodeJS 10 версии и выше.
Проверяете работу команды node и npm в командной строке Windows (cmd).

1. Извлекаете папку из архива
2. Открываете в командной строке папку (например, cd c:\documents\web_application
3. В командной строке прописываете поочередно:

npm install
npm run build
npm start

Если все в порядке, командная строка не выдаст ошибок. Откроется сайт на http://localhost:5000.
